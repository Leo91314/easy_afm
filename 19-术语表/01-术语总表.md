收集本知识库常见术语，用最简单语言解释，随时可回查。

- 常见缩写和英文术语中文含义

- 无，遇到看不懂就来查

## 正文

| 术语 | 全称/英文 | 通俗解释 | 在项目里对应 |
|---|---|---|---|
| AFM | Atomic Force Microscope | 用探针“摸表面”的显微方法 | 项目目标系统 |
| OPU | Optical Pick-Up Unit | 光学拾取模块来源 | BUILD_GUIDE 提及关键材料 |
| Probe | 探针 | 与样品交互的尖端部件 | 光学与扫描核心 |
| Cantilever | 悬臂梁 | 承载探针的微小弹性结构 | AFM 检测关键 |
| PCB | Printed Circuit Board | 电路板 | `pcb/` 目录 |
| BOM | Bill of Materials | 物料清单 | 采购和装配基础 |
| Gerber | - | 打板生产文件 | 需确认是否导出 |
| PnP | Pick-and-Place | 贴片坐标文件 | SMT 生产相关 |
| MCU | Microcontroller Unit | 微控制器 | 本项目与 ESP32 相关 |
| ESP32 | - | 常见 Wi-Fi/蓝牙微控制器 | `platformio.ini` 目标板相关 |
| PlatformIO | - | 固件开发工具链 | `firmware/` 编译烧录 |
| Serial | 串口通信 | 电脑和板子对话通道 | GUI 与固件通信 |
| Baudrate | 波特率 | 串口通信速度 | 代码中常见 115200 |
| GUI | Graphical User Interface | 图形界面 | `gui/afm_gui.py` |
| PyQt6 | - | Python 图形界面库 | GUI 依赖 |
| pyqtgraph | - | 实时绘图库 | GUI 图像显示 |
| ADC | Analog-to-Digital Converter | 模数转换器 | 采集模拟信号 |
| DAC | Digital-to-Analog Converter | 数模转换器 | 输出控制电压 |
| PID | Proportional-Integral-Derivative | 闭环控制算法 | 固件与 GUI 中有参数 |
| Piezo Actuator | 压电执行器 | 施加电压产生微位移 | 精调运动核心 |
| Stepper Motor | 步进电机 | 按步进控制位置的电机 | 粗调运动 |
| Focus | 聚焦流程 | 找到较优检测状态 | GUI 功能之一 |
| Approach | 接近流程 | 让探针逐步接近样品 | GUI/固件流程 |
| Scan | 扫描流程 | 逐点采集并成图 | GUI/固件流程 |
| TIFF | Tagged Image File Format | 常见科学图像格式 | GUI 导出功能 |
| STEP | .step 文件 | 通用 3D 几何文件 | CAD 结构件 |
| 3MF | .3mf 文件 | 3D 打印打包格式 | CAD/打印流程 |
| F3D/F3Z | Fusion 360 文件 | 设计源文件/归档 | CAD 深度编辑 |
