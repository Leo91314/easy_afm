## 正文
常用命令（先理解用途，不要求背）：
```bash
git clone <仓库地址>
git pull
git status
git log --oneline -n 5
```

建议习惯：
- 每次改动前先 `git status`
- 每次学习结束记录当天结论

## 用最简单的话再说一遍
