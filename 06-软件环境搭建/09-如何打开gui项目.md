# 如何打开gui项目

## 这一页是干什么的
教你启动 GUI 并做第一次界面验证。

## 你会学到什么
- GUI 启动命令
- 启动后应检查哪些区域

## 先决条件
- [[06-软件环境搭建/05-安装PyQt6等依赖]]

## 预计耗时
- 20 分钟

## 正文
在 `red-panda-afm/gui` 目录（激活 `.venv`）执行：
```bash
python afm_gui.py
```

启动后检查：
- 主窗口是否打开
- Focus/Approach/Scan 子窗口按钮是否可点
- 程序是否立刻崩溃

## 用最简单的话再说一遍
看到 GUI 主窗口，就是软件环境的关键里程碑。

## 在 red-panda-afm 项目里它对应什么
- GUI 入口：`afm_gui.py`

## 这一页完成后，你应该能做到什么
- 成功打开 GUI

## 常见误区
- 在未激活虚拟环境时运行
